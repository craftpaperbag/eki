# 駅コンパス

スマートフォンで現在地から最寄り駅までの距離と方角を表示する Next.js (App Router) アプリです。位置情報と端末の方位センサーをブラウザ経由で取得し、すべてクライアントサイドで処理します。

## セットアップ

```bash
npm install
npm run dev
```

環境変数の例は `.env.example` を参照してください。クライアントで使う値は `NEXT_PUBLIC_` プレフィックスを付けてください。

## データソース

JR 駅の座標データは [JR 駅データ/路線LOD](https://uedayou.net/jrslod/) を元にし、`src/data/stations.json` に静的同梱しています（本リポジトリではプレビュー用に主要駅のみ抜粋しています）。

## 開発メモ
- サーバーレス関数は使用せず、10 秒以上かかる処理を避けています。
- 画像最適化回数を節約するため、`next/image` ではなく標準の `<img>`/CSS を利用しています。
- TypeScript と ESLint を有効にしてビルドエラーを未然に防ぎます。
